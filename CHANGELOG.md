## [1.0.0] — 2025-07-02

### Added
- Initial release as **AIObar**
- Social share buttons: Facebook, LinkedIn, X (Twitter), Reddit
- Summarize dropdown: Claude, ChatGPT, Perplexity — pre-filled with summarise prompt
- Copy for LLM — Title + URL + plain-text body in one click
- View Markdown collapsible panel
- All visual design configurable via `AIOBAR_*` PHP constants
- `PUBLIC_BASE_URL` for subdomain/reverse-proxy URL rewriting
- Desktop flexbox layout + mobile CSS Grid
- All JS via `wp_enqueue_script()` — no inline scripts in content
- Document-level event delegation
- Zero external dependencies
